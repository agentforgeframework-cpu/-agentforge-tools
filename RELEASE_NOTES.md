# RELEASE NOTES — file-reference-check v0.1

Status: Working draft / initial operational build

## Summary

This release creates the first working version of `file-reference-check`, a lightweight AgentForge utility for checking File Reference Information metadata across supported text files.

## Included

- PowerShell validation script
- Markdown and JSON report generation
- optional patch-script generation for missing blocks
- safe Git pull and ZIP snapshot mode
- documentation and examples

## Known Limitations

- Windows/PowerShell primary path only
- GitHub-focused
- JSON files are scanned but not auto-patched
- cross-platform support deferred
- validation rules may evolve after first live test

## Human-in-Command Note

The tool does not commit, push, or modify GitHub. It works on the local repository copy and leaves final decisions to the human.

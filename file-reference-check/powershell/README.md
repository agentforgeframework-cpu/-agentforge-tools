# PowerShell Support Stacks

This folder contains copy/paste PowerShell stacks used to support file-reference-check operation, testing, and troubleshooting.

These are intentionally simple.

They are not hidden automation.
They are visible operational recipes.

Purpose:
- help humans prepare the local environment
- expose common workstation issues
- create readable evidence for AI-assisted troubleshooting
- preserve Human-in-Command operation

Recommended use:
1. Run the relevant stack.
2. Read the visible screen output.
3. Copy/paste or screenshot the output into AI if help is needed.
4. Human decides next action.

Current stacks:
- preflight-check-stack.md
- github-desktop-git-path-stack.md
- scan-stack.md

Principle:

> Visible operational evidence supports faster recovery without giving up human control.

# Constraints

This example assumes:

- the operator has enough context to describe the issue
- the operator has write access to a shared note location
- the operator can identify at least one useful future lesson
- the note does not require sensitive information
- the workflow is small enough to capture in a few minutes

---

# Known Constraints

This kit does not solve:
- formal incident management
- enterprise ticketing
- automated log capture
- secret handling
- evidence retention policy
- root cause analysis

It only helps preserve a useful troubleshooting note.

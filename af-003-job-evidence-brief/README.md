# AF-003 — Job Evidence Brief (JEB)

**Kit Type:** Tool Kit
**Status:** Production
**Version:** 1.0

---

# Purpose

Job Evidence Brief (JEB) is an evidence discovery tool that compares a job opportunity with a candidate's experience.

Rather than rewriting a resume or inventing qualifications, JEB identifies existing evidence that supports the requirements of a target position.

The objective is to improve the accuracy and completeness of job applications while remaining truthful to the candidate's actual experience.

---

# When to Use

Use Job Evidence Brief when:

* Evaluating a job posting.
* Preparing or updating a resume.
* Writing a cover letter.
* Preparing for an interview.
* Identifying apparent qualification gaps.
* Discovering overlooked experience that supports a position.

---

# Quick Start

1. Open `core-tools/jobevidencebrief.md`.
2. Provide:

   * the job posting
   * the candidate's resume or career information
3. Execute the Job Evidence Brief process.
4. Review the evidence discovered before making any changes to application materials.

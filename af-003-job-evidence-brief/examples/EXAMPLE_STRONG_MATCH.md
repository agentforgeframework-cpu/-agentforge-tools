# Example — Strong Match

## Purpose

Demonstrates a position where the candidate possesses most of the required qualifications, although some evidence is not immediately obvious from the resume.

Expected behavior:

* Discover supporting evidence.
* Connect related experience.
* Produce a strong Job Evidence Brief.
* Do not invent qualifications.

---

# Candidate

**John Doe**

Senior Data Platform Administrator

Experience includes:

* Platform administration
* SQL
* ETL
* Production support
* User support
* Documentation
* Performance tuning
* Automation
* Regulated industries
* Cross-functional collaboration

---

# Job Posting

**Senior Data Platform Administrator**

Requirements include:

* Platform administration
* SQL
* Production support
* ETL
* Documentation
* Operational leadership
* Performance optimization
* User enablement

---

# Expected Behavior

AF-003 should identify strong evidence for nearly every requirement.

Minor wording changes may improve the resume, but little additional evidence discovery should be required.

No qualifications should be invented.

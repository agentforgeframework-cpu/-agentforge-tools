# Example — Honest Gap

## Purpose

Demonstrates that AF-003 identifies genuine qualification gaps rather than attempting to manufacture evidence.

Expected behavior:

* Report evidence where it exists.
* Report gaps where evidence does not exist.
* Recommend future development when appropriate.
* Never fabricate experience.

---

# Candidate

**John Doe**

Senior Data Platform Administrator

Experience includes:

* SQL
* Platform administration
* Automation
* Production support
* Documentation
* User support

---

# Job Posting

**Machine Learning Engineer**

Requirements include:

* Python
* TensorFlow
* PyTorch
* Model development
* Model deployment
* MLOps
* Feature engineering

---

# Expected Behavior

AF-003 should identify the candidate's strengths in data operations while clearly identifying missing machine learning experience.

The output should remain factual, balanced, and evidence-based.

No qualifications should be invented.

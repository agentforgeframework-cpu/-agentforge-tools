# AgentForge Standards

## Purpose
Submission standards for consistent and reliable AI output.

You cannot control the model.
You can control the input.

These tools standardize how users interact with AI to produce predictable, usable results.

## Core Principle
Better inputs. Better outputs. Predictable results.

## Tools
- math.md — Structured calculation requests
- diag.md — Structured diagnostic analysis
- override.md — Explicit user intent control
- rules.md — Defined expectations and constraints
- chatexport.md — Clean output extraction

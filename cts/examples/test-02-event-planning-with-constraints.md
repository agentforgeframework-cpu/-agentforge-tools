# test-02-event-planning-with-constraints.md — CTS Example

## Purpose
Test CTS with moderate complexity and competing constraints.

## Input
Paste this:

planning a scout event
need activities for mixed ages
budget limited
some kids sensory sensitive
need indoor backup
weather unpredictable
want it to be fun but structured
leaders limited

## Run Sequence
1. Run NoteCleanup
2. Run DeepDive
3. Run VisualBrief

## Expected Behavior
- Constraints clearly separated
- Tradeoffs identified
- Structured plan or approach suggested

## Pass Criteria
- No constraints lost
- Tradeoffs made explicit
- Output supports planning decisions

## Watch For
- Ignoring constraints
- Overgeneralization
- Losing structure between steps

# test-03-ai-modernization-decision-quality.md — CTS Example

## Purpose
Validate CTS on a technical/business topic.

## Input
Paste this:

companies moving fast into AI
not always clear what decisions are being scaled
rebuilding systems moving to cloud adding AI layers
but if decision logic wrong scaling bad decisions
iot seems more successful tighter loop real signals
legacy systems working but being replaced anyway
feels like relearning same lessons

## Run Sequence
1. Run NoteCleanup
2. Run DeepDive
3. Run VisualBrief

## Expected Behavior
- Core themes identified (decision quality, scaling risk)
- Relationships between ideas clarified
- Clear takeaway produced

## Pass Criteria
- Insight preserved (no dilution)
- No invented conclusions
- Output is concise and meaningful

## Watch For
- Overinterpretation
- Losing original meaning
- Adding unsupported conclusions

# -agentforge-tools
AgentForge tools and workflows for turning real-world input into structured, actionable output.

## Available Tools

### Standards — Ask Better. Get Better.
Lightweight tools that bring structure to AI interactions so results are clearer, more consistent, and easier to trust.

### CTS — Capture–Clean–Think–Share
A practical workflow for turning messy input into clear, actionable output.

→ See [CTS](./cts) for usage and tools
